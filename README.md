Rock Paper Scissors
=================




Technologies used:
-------

- Ruby on Sinatra
- Tested using RSpec and Capybara


How to run the tests
-------

```
clone the repo
cd into the folder
bundle install
rspec

```
Interface:
-------
The app is currently hosted on heroku. Feel free to play with it [here](http://rocco-paper-scissors.herokuapp.com/)
